# Todo API Bun

Api para cadastro e listagem de tarefas com o objetivo de aprender e utilizar Bun

Tecnologias usadas:
- Bun (1.2.4)
- Drizzle ORM
- Postgres
- Nginx
- Docker
- Typescript

```bash
bun install
```
Para configurar o banco de dados
```bash

```
Para rodar o projeto
```bash
bun run dev
```
